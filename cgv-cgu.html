3. Prestations proposées

3.1. Offres standards automatisées

La plateforme Vitrine Express permet à l’Utilisateur d’accéder à des solutions digitales clés en main.

  • Création de pages web via le configurateur Vitrine Express
  • Kits de visuels LinkedIn et ressources digitales accessibles directement depuis la plateforme
  • La modifications gratuite d'une Vitrine générée via le configurateur Vitrine Express, dans la limite d’une mise à jour toutes les 72 heures.

Les caractéristiques essentielles de chaque offre sont présentées sur le site.

3.2. Prestations sur-mesure

Design web personnalisé, rebranding LinkedIn, copywriting ou toute prestation spécifique faisant l’objet d’un devis préalable.

3.3. Connexion à un nom de domaine personnalisé

Une option de connexion de nom de domaine pour les pages web générées via le configurateur Vitrine Express peut être proposée lors du paiement.

  • Utilisateur disposant déjà d’un nom de domaine : l’Utilisateur fournit les accès nécessaires pour permettre la configuration technique.
  • Utilisateur ne disposant pas de nom de domaine : AM 2.0 peut procéder à l’achat du nom de domaine pour le compte de l’Utilisateur, pour une durée initiale d’un (1) an, celui-ci étant enregistré au nom de l’Utilisateur.

La mise en place technique est conditionnée à la transmission par l’Utilisateur des informations et accès nécessaires.

Le renouvellement du nom de domaine après la première année reste à la charge exclusive de l’Utilisateur.

4. Tarifs et Paiement

4.1. Offres Standards (Plateforme)

Les tarifs des offres Vitrine Express accessibles via la plateforme automatisée sont ceux indiqués sur le site au moment de la configuration. Le montant total est confirmé sur la page de paiement Stripe. Le paiement est intégral et conditionne l'accès au téléchargement ou à la mise en ligne.

4.2. Prestations Sur-Mesure (Design & Copywriting)

Pour tout projet spécifique (Design exclusif, Rebranding LinkedIn, option Copywriting), les tarifs sont communiqués par devis personnalisé après échange initial.

  • Validation : La commande est validée dès le règlement via le lien de paiement sécurisé (Stripe) fourni avec le devis.
  • Design exclusif : Le paiement du design sur-mesure déclenche l'envoi d'une référence d'accès permettant de charger le modèle sur le configurateur. Le paiement final de la "Vitrine" reste dû pour la mise en ligne.
  • Rebranding LinkedIn : Le paiement s'effectue par étapes (Design, puis solde Copywriting) via liens Stripe dédiés.

4.3. Option connexion au nom de domaine personnalisé

Le prix de l’option nom de domaine couvre uniquement l’achat initial et la configuration technique. Les frais de renouvellement annuel ne sont pas inclus sauf mention contraire dans le devis.

La validation du paiement vaut acceptation pleine et entière des présentes CGV/CGU. Les livrables numériques sont fournis immédiatement après validation du paiement, sauf indication contraire dans le devis pour les prestations sur-mesure.

Les délais de réalisation pour les prestations personnalisées sont communiqués au client lors de la commande.

4.4. Livraison et accès aux contenus

Pour les prestations automatisées via le configurateur Vitrine Express, la livraison est considérée effectuée dès que l’Utilisateur a :

  • été redirigé vers la page de succès après le paiement ;
  • cliqué sur le bouton de confirmation sur la page de succès, déclenchant la mise en ligne automatique de sa page web et le téléchargement de son fichier HTML source ;
  • pour les kits LinkedIn, téléchargé les visuels JPEG et le fichier HTML source associés.

L’Utilisateur peut utiliser le fichier HTML source pour apporter des modifications à sa page web ou son kit LinkedIn via la fonctionnalité « Modifier ma vitrine » du configurateur. Les modifications sont possibles gratuitement, dans la limite d’une mise à jour toutes les 72 heures, sous réserve que le fichier HTML source fourni n'ait pas été corrompu ou modifié.

En cochant la case prévue avant le paiement, l’Utilisateur reconnaît et accepte que ses produits numériques seront livrés immédiatement après paiement et renonce expressément à son droit de rétractation conformément aux dispositions légales applicables.

Pour les prestations sur-mesure :

  • Design exclusif : le paiement de l’option design sur-mesure déclenche l’envoi d’une référence d’accès permettant de charger le modèle sur le configurateur. La livraison finale de la "Vitrine" se fait ensuite comme pour toute autre vitrine du configurateur. La référence est fournie par mail ou messagerie privée (LinkedIn, Facebook, etc.) dans le délai indiqué sur le devis.
  • Copywriting : les documents sont transmis au format PDF par mail ou messagerie privée dans le délai indiqué sur le devis.
  • Rebranding LinkedIn : les visuels sont transmis via lien Canva par mail ou messagerie, et le copywriting de la bio et de la tagline est fourni au format PDF dans le délai indiqué sur le devis.

Pour les prestations sur-mesure, le droit de rétractation ne s’applique pas, étant donné que la prestation est considérée comme exécutée dès que le contenu ou la référence est transmis à l’Utilisateur dans le délai indiqué sur le devis, conformément aux dispositions légales applicables.
